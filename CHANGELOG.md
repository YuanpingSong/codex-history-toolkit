# Changelog

All notable changes to this project are documented here.

## 0.3.0 — 2026-07-13

First public release under the **Codex History Toolkit** name.

- Added private, content-minimizing audits of local Codex history.
- Added conservative classification for interactive, automated, guardian,
  subagent, empty-shell, and ambiguous records.
- Added explicit custom automation originators without project-specific rules.
- Added immutable archive planning plus verified apply and resume workflows.
- Added private interactive-history export bundles for migration to a new Mac.
- Added malformed-ID and structural anomaly reporting.
- Added generated-data guards, private artifact permissions, CI, packaging,
  security guidance, and an MIT license.
